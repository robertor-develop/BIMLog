Getting Started
Up and running in 6 steps
From account creation to fully coordinated BIM project — here's the complete workflow.

01
Create your account
Register with your name, email, and company. Your account is immediately ready — no email confirmation required.

Use your company email
Company name appears in reports
02
Create a project
Click '+ New Project' on the dashboard. Give it a name and a short code (e.g. NYC-270). You are automatically assigned Project Admin.

Code must be uppercase alphanumeric
Invite team from the Team tab
03
Define naming conventions
Go to Convention Builder (admin only). Add fields like Originator, Discipline, Phase. Activate when ready — all uploads validate against it instantly.

Set allowed values per field
Activate/deactivate without deleting
04
Upload & validate files
Open the Files tab and upload BIM files. The server validates the filename against the active convention. Non-compliant files are rejected with field-level errors.

Use Name Generator for compliant names
See exact field violations on rejection
05
Manage RFIs & Submittals
Create and track Requests for Information and Submittals from their dedicated tabs. Assign status, reference files, and keep the full coordination trail.

Status follows ISO 19650 workflow
All changes logged automatically
06
Monitor & report
Use the Analytics dashboard for a live overview of files, RFIs, and team activity. The Activity Log is immutable — every action is permanently recorded.

Activity log cannot be deleted
Switch language EN / ES anytime
